# Contributing to Airsoft Display

Thank you for your interest in this project! This is a personal embedded hardware project shared for educational purposes.

## Support Expectations

This is a hobby project maintained on a best-effort basis. While issues and pull requests are welcome, responses may be delayed or limited. Please understand that:

- **No guaranteed support** - Issues may not receive immediate attention
- **Hardware-specific** - This project is designed for a specific airsoft rifle setup
- **Limited scope** - Feature requests outside the core shot counter/battery monitor functionality may not be implemented

## How to Contribute

If you'd like to contribute:

1. **Open an issue first** to discuss significant changes
2. **Fork the repository** and create a feature branch
3. **Follow the existing code style** (see `.github/copilot-instructions.md`)
4. **Test on hardware** before submitting pull requests
5. **Document your changes** in commit messages and code comments

## Hardware Safety Disclaimer

**⚠️ WARNING: This project involves working with lithium polymer (LiPo) batteries and custom electronics.**

- **LiPo batteries are dangerous** if mishandled - fire and explosion risk
- **Use proper voltage regulation** - incorrect wiring can damage components or cause fires
- **Test circuits carefully** before connecting to expensive airsoft equipment
- **Know your local laws** regarding airsoft modifications
- **No warranties** - you build and use this project at your own risk
