# Datasheet Drop Folder

This directory is reserved for third-party datasheets, reference manuals, or vendor utilities that should **not** be committed to the public repository. The files you place here are ignored by Git so you can keep local copies without redistributing copyrighted material.

Place any downloaded copies directly in this folder for your own reference. If new documentation is required, update this table instead of adding the actual files to version control.
